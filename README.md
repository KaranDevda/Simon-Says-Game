# 🎮 Simon Says Game

A classic, interactive memory game built with JavaScript, HTML, and CSS. Test your memory by repeating an increasingly complex sequence of colors and sounds!

## 🎯 How to Play
1. Click the **Start** button (or press any key) to begin the game.
2. The game will highlight a color. 
3. Click the same color to pass the round.
4. In the next round, the game will add a new color to the sequence.
5. You must repeat the **entire sequence** in the exact order.
6. Make one mistake, and the game is over! Your highest score will be displayed.

---

## 🛠️ Technologies Used
* **HTML5:** Structures the game board and buttons.
* **CSS3:** Styles the UI, grid layout, and button flash animations.
* **JavaScript (ES6):** Handles the game logic, sequences, event listeners, and user inputs.
